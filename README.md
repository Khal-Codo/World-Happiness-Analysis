# World Happiness Data Analysis — Excel Study (2020–2024)

## Overview

This project analyzes multi-year world happiness data to identify regional trends and support hypothetical business expansion decisions. The analysis was conducted using Microsoft Excel with structured charts and comparative scoring.

## Objectives

* Analyze multi-year happiness indicators
* Compare regional performance patterns
* Identify favorable expansion regions
* Produce decision-support visuals and report findings

## Tools Used

* Microsoft Excel
* Spreadsheet modeling
* Charting and visual analysis

## Artifacts Included

* Source dataset (2020–2024)
* Excel analysis workbook
* Generated charts
* Final written report

## Methods Used

* Comparative trend analysis
* Regional scoring comparison
* Visual interpretation

## Value Demonstrated

Demonstrates applied data analysis, structured interpretation, and decision-support reporting using spreadsheet tools.
